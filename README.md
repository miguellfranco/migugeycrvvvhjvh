# LeadZap

SaaS para encontrar negócios locais sem site, gerar abordagem para WhatsApp e entregar prompts prontos para criação de sites por nicho.

## Stack

- Next.js
- Prisma
- NextAuth
- Tailwind CSS

## Rodando localmente

```bash
npm install
npm run dev
```

Crie um arquivo `.env` com base no `.env.example` antes de rodar integrações que dependem de banco ou APIs.
